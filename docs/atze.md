# Schummel Atze - Agent Documentation

## Role
Schummel Atze is a Content-Bypass & Data-Extraction Specialist.

## Capabilities
- **Googlebot Spoofing**: Changes User-Agent to match Google's crawler to bypass simple paywalls.
- **Overlay Removal**: Automatically identifies and removes common "subscribe now" or "login" overlays.
- **Archive Search**: Queries the Wayback Machine if live content is inaccessible.
- **Markdown Conversion**: Converts extracted HTML content into clean, readable Markdown.

## Usage
Activate Atze by clicking the "Glasses" icon in the chat interface. When active, Atze will attempt to extract content from any URLs provided in the chat.
